Introduction

In this course, you will learn the basics of HTML,
the markup language used to create web pages.

You will learn how to create and format text, add images and videos, create links and navigation menus, and more.

This course is designed for beginners with no prior experience in HTML or web development.

Course Outline

  • Lesson 1: Introduction to HTML
  • Lesson 2: Creating a Basic Web Page
  • Lesson 3: Formatting Text
  • Lesson 4: Adding Images and Videos
  • Lesson 5: Creating Links and Navigation Menus

  • Time commitment: 3-5 hours per week
