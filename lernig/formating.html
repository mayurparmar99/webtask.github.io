HTML Formatting Tags

BOLD TAG :-

<b></b>

OUTPUT :-

This text will be bold

ITALIC TAG :-

<i> </i>

OUTPUT :-

This text will be Italic

UNDERLINE TAG :-

<u> </u>

OUTPUT :-

This text will be underline

STRIKE TAG :-

<strike> </strike>

OUTPUT :-

This text will be underline

SUPERSCRIPT TAG :-

<sup> </sup>

OUTPUT :-

E=(x+y)2

SUBSCRIPT TAG :-

<sub> </sub>

OUTPUT :-

H2O

EMPHASIZED TAG :-

<em> </em>

OUTPUT :-

This text will be emphasized and bold

STRONG TAG :-

<strong> </strong>

OUTPUT :-

This text will be strong

This text will be smaller

This text will be bigger
