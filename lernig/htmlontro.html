What is HTML?

HTML stands for HyperText Markup Language. It is a markup language used to create web pages.

HTML Basics

HTML documents are made up of elements, which are defined by tags. Tags are enclosed in angle brackets, like this:

<tagname>content</tagname>

The opening tag is the first part of the element, and the closing tag is the second part of the element. The content is the text or other elements contained within the tags.

HTML Elements

HTML elements can be divided into two categories: block-level elements and inline elements. Block-level elements start on a new line and create a rectangular block of content. Inline elements do not start on a new line and only take up as much space as necessary to display their content.

Some common block-level elements include:

  • <h1> - <h6>: heading elements
  • <p>: paragraph element
  • <div>: division element used for grouping other elements together
  • <ul>: unordered list element
  • <ol>: ordered list element
  • <li>: list item element

Some common inline elements include:

  • <a> - anchor tag used to create hyperlinks
  • <span> - generic inline container element that can be used for grouping inline elements
  • <strong> - used to indicate strong importance
  • <em> - used to indicate emphasis
  • <img> - used to embed images into a document
  • <input> - used to create form controls such as text boxes, checkboxes, and radio buttons
  • <br> - line break element used to force a line break in text
  • <sup and sub> - used to create superscript and subscript text respectively
  • <code> - used to indicate computer code or programming language syntax within a document
  • <cite> - used to indicate a citation or reference to a source
